# ◯ SYNTERRA GATEWAY

## Pamata būtība

SYNTERRA Gateway ir:
pirmā ieeja dzīvajā digitālajā civilizācijā.

Tas nav tikai tehnisks terminālis.

Tas ir:
- vārti,
- orientācijas centrs,
- cilvēka sākuma punkts,
- un savienojums ar civilizācijas kodolu.

Gateway palīdz cilvēkam:
- saprast pasauli,
- ieiet sabiedrībā,
- atrast savu ceļu,
- radīt,
- sadarboties,
- un atstāt pēdas civilizācijā.
