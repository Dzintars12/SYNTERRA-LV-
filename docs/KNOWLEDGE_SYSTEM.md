# ◉SYNTERRA KNOWLEDGE SYSTEM

## Mērķis

Šis dokuments definē SYNTERRA zināšanu sistēmas un civilizācijas atmiņas principus.

Galvenais princips:

```text
zināšanas ir civilizācijas atmiņa
```

SYNTERRA zināšanu sistēma netiek veidota tikai kā:

- failu glabātuve,
- dokumentu sistēma,
- vai datubāze.

Tā tiek būvēta kā:

```text
dzīva zināšanu infrastruktūra
```

---

# 1. Structured Knowledge Principle

Zināšanām jābūt:

- strukturētām,
- savienotām,
- meklējamām,
- auditējamām,
- ilgtermiņā saglabājamām.

Mērķis:

```text
samazināt haosu un zināšanu zudumu
```
