# ◉SYNTERRA PLATFORMU UN SPĒĻU SINTĒZES ANALĪZE

## Mērķis

Šis dokuments definē, kā SYNTERRA pāriet no konstitūcijas līmeņa uz reālu būvniecību.

Galvenais princips:

```text
neko nekopēt
visu analizēt
vērtīgo sintezēt
```

SYNTERRA nedrīkst kļūt par:

- Facebook kopiju,
- Discord kopiju,
- Roblox kopiju,
- Minecraft kopiju,
- Fortnite kopiju,
- TikTok kopiju,
- vai kādas citas platformas atdarinājumu.

SYNTERRA uzdevums ir:

```text
izanalizēt esošās platformas
saprast to plusus un mīnusus
un sintezēt jaunu harmonisku virzienu
```

---

# 1. Pamatprincips

Ja 5 gadīgs bērns intuitīvi nevar saprast:

- kur viņš atrodas,
- ko drīkst darīt,
- kas notiek tālāk,
- kur ir cilvēki,
- kur ir palīdzība,
- kā sākt radīt,

tad dizains vēl nav pietiekami labs.

SYNTERRA būvniecības princips:

```text
vienkārša virsma
ļoti dziļš kodols
```

# 2. Kāpēc Jāanalizē Citas Platformas

Populārās platformas satur ļoti vērtīgas mācības.

Tās parāda:

- kas cilvēkiem patīk,

- kur rodas radošums,
- kur rodas ekonomika,
- kur rodas sociālie mezgli.

Taču SYNTERRA nedrīkst pārņemt citu platformu kļūdas.

# 4. Ko SYNTERRA Var Mācīties

## No Roblox

Pozitīvais:

- user-generated worlds,
- creator tools,
- social play,
- jauniešu intuitīva iesaiste,
- milzīga mērogojamība.

Riski:

- bērnu drošība,
- monetizācijas spiediens,
- content moderation,
- haotiska kvalitāte.

SYNTERRA secinājums:

```text
radošuma rīki ir spēcīgi
bet tiem jābūt savienotiem ar drošību un morālo kodolu
```

---

## No Discord

Pozitīvais:

- kopienu serveri,
- balss komunikācija,
- ātra sadarbība,
- tematiskas grupas,
- organiskas kopienas.

Riski:

- informācijas haoss,
- fragmentācija,
- moderācijas grūtības,
- privātuma un drošības riski.

SYNTERRA secinājums:

```text
komunikācijai jābūt strukturētai
nevis tikai kanālu haosam
```

---

## No TikTok / Instagram

Pozitīvais:

- ļoti vienkāršs onboarding,
- intuitīva lietošana,
- ātra satura atklāšana,
- vizuāla pievilcība.

Riski:

- atkarība,
- uzmanības ekspluatācija,
- virspusēja socializācija,
- algoritmiska manipulācija.

SYNTERRA secinājums:

```text
vienkāršība ir jāņem
bet manipulācija jāatmet
```

---

## No Minecraft

Pozitīvais:

- brīvība radīt,
- vienkārša sākuma mehānika,
- dziļa radošuma telpa,
- kopiena,
- mācīšanās caur darbību.

Riski:

- vāja sociālā struktūra bez ārējiem rīkiem,
- ierobežota organizāciju infrastruktūra.

SYNTERRA secinājums:

```text
radošuma brīvība jāapvieno ar civilizācijas struktūru
```

---

## No GitHub

Pozitīvais:

- version control,
- collaboration,
- contribution history,
- open-source kultūra,
- dokumentācijas disciplīna.

Riski:

- sarežģīts iesācējiem,
- nav intuitīvs bērnam,
- pārāk tehnisks plašai sabiedrībai.

SYNTERRA secinājums:

```text
GitHub dziļums jāapvieno ar bērnam saprotamu virsmu
```
