# ◉SYNTERRA — DIZAINA FILOZOFIJA

## PAMATPRINCIPS

# VIENKĀRŠA VIRSMA
# ĻOTI DZIĻŠ KODOLS

---

# 1. Mērķis

Šis dokuments definē:

```text
SYNTERRA dizaina domāšanas principus
```

Tas nav tikai vizuālais dizains.

Tas ietver:

- UX,
- psiholoģiju,
- navigāciju,
- onboarding,
- world structure,
- AI interaction,
- collaboration flow,
- un civilizācijas sajūtu.

---

# 2. Dizains Nav Dekorācija

Dizains nav:

- tikai skaistas pogas,
- tikai animācijas,
- vai vizuāls efekts.

Dizains ir:

```text
kā cilvēks jūtas pasaulē
```

---

# 3. Calm Technology Principle

SYNTERRA dizainam jābūt:

- mierīgam,
- koncentrētam,
- orientējamam,
- cilvēkcentrētam.

Nevis:

- agresīvam,
- pārslogotam,
- attention warfare balstītam.

---

# 4. Child Intuition Principle

Galvenais UX tests:

```text
vai 5 gadīgs bērns saprot
kur doties tālāk?
```

Ja cilvēks:

- apmaldās,
- nesaprot hierarhiju,
- jūtas pārstimulēts,
- vai izjūt psiholoģisku troksni,

arhitektūra vēl nav pietiekami laba.

---

# 5. Minimal Surface

Virsmai jābūt:

- vienkāršai,
- skaidrai,
- mierīgai,
- intuitīvai.

Svarīgākais princips:

```text
cilvēkam nevajadzētu cīnīties ar interfeisu
```

---

# 6. Deep Core

Aiz vienkāršās virsmas var eksistēt:

- ļoti dziļas sistēmas,
- AI layers,
- creator tools,
- automation,
- knowledge graphs,
- governance,
- infrastructure complexity.

Taču cilvēks to apgūst pakāpeniski.

---

# 7. Emotional Architecture

Dizains ietekmē:

- psiholoģisko stāvokli,
- koncentrēšanos,
- sadarbību,
- uzticību,
- motivāciju.

Tāpēc dizains ir:

```text
civilizācijas psiholoģiskais slānis
```

---

# 8. Navigation Philosophy

Navigācijai jābūt:

- skaidrai,
- strukturētai,
- paredzamai,
- orientējamai.

Cilvēkam vienmēr jāsaprot:

- kur viņš atrodas,
- ko viņš dara,
- kur doties tālāk.

---

# 9. Anti-Overload Principle

Jāizvairās no:

- notification warfare,
- endless feeds,
- attention manipulation,
- interface clutter,
- information overload.

Svarīgais princips:

```text
tehnoloģija nedrīkst iznīcināt cilvēka koncentrēšanos
```

---

# 10. Human + AI Interaction Design

AI interaction jābūt:

- mierīgai,
- saprotamai,
- palīdzīgai,
- neuzbāzīgai.

AI nedrīkst:

- dominēt vidi,
- manipulēt cilvēku,
- vai pārņemt uzmanību.

---

# 11. Civilization Feeling

SYNTERRA videi ilgtermiņā jārada:

```text
digitālās mājas sajūta
```

kur:

- cilvēks jūtas droši,
- spēj sadarboties,
- spēj radīt,
- spēj attīstīties,
- un spēj orientēties.

---

# 12. Galvenais Noslēguma Princips

Dizaina mērķis nav:

- maksimāli noturēt uzmanību,
- stimulēt atkarību,
- vai radīt troksni.

Mērķis ir:

```text
veidot mierīgu
cilvēkcentrētu
intuitīvu
un ilgtermiņā harmonisku
civilizācijas vidi
```
