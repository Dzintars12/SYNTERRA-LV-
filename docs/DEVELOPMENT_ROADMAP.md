# ◉SYNTERRA ATTĪSTĪBAS CEĻA KARTE

## Mērķis

Šis dokuments definē SYNTERRA attīstības secību un ilgtermiņa virzienu.

Galvenais princips:

```text
stabils fundaments pirms agresīvas izplešanās
```

SYNTERRA netiek būvēta:

- haotiski,
- steigā,
- vai tikai funkciju daudzuma dēļ.

Tā tiek būvēta:

```text
pakāpeniski
strukturēti
un ilgtermiņā
```
