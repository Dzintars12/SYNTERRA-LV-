# ◉SYNTERRA FIRST CONTACT AND ONBOARDING ARCHITECTURE

## Mērķis

Šis dokuments definē:

```text
cilvēka pirmo kontaktu ar SYNTERRA pasauli
```

Tas ir viens no kritiskākajiem civilizācijas arhitektūras slāņiem.

Jo:

```text
pirmās 30 sekundes nosaka
vai cilvēks jūtas droši
vai saprot vidi
un vai vēlas palikt
```

---

# 1. Galvenais Princips

Pirmajam kontaktam jābūt:

- mierīgam,
- intuitīvam,
- saprotamam,
- cilvēkcentrētam,
- bez haosa.

SYNTERRA nedrīkst:

- pārstimulēt,
- radīt troksni,
- uzreiz pārslogot cilvēku,
- vai uzbrukt ar funkcijām.

---

# 2. Bērna Intuīcijas Tests

Galvenais tests:

```text
vai 5 gadīgs bērns intuitīvi saprot
kas notiek tālāk?
```

Ja cilvēks:

- apmaldās,
- nesaprot vidi,
- jūtas psiholoģiski nospiests,
- vai nezina kur doties,

onboarding vēl nav pietiekami labs.

---

# 3. First Contact Philosophy

Pirmajam kontaktam jārada:

```text
mierīgas digitālās pasaules sajūta
```

Nevis:

- agresīva aplikācija,
- reklāmu vide,
- vai attention economy mehānisms.

Cilvēkam jājūt:

- orientācija,
- drošība,
- skaidrība,
- potenciāls,
- sadarbības iespēja.

---

# 4. Pirmā Ieeja

Pirmajai ieejai jābūt:

- ļoti vienkāršai,
- vizuāli mierīgai,
- intuitīvai.

Sākotnējie elementi:

- valodas izvēle,
- guest mode,
- account creation,
- AI Guide palīdzība.

---

# 5. Guest Mode

Cilvēkam jāspēj:

```text
ieiet pasaulē bez bailēm un bez spiediena
```

Guest mode palīdz:

- izpētīt vidi,
- saprast pasauli,
- iepazīt social layer,
- redzēt creator spaces,
- sajust civilizācijas atmosfēru.

---

# 6. AI Guide

AI Guide nav:

- agresīvs assistants,
- reklāmas mehānisms,
- vai manipulācijas instruments.

AI Guide ir:

```text
mierīgs civilizācijas gids
```

Tas palīdz:

- orientēties,
- saprast pasauli,
- atrast cilvēkus,
- atrast projektus,
- sākt radīt.

---

# 7. World Hub

Pirmajai telpai jābūt:

- skaidrai,
- mierīgai,
- minimālistiskai,
- strukturētai.

World Hub kļūst par:

```text
civilizācijas centrālo ieejas telpu
```

Tur nedrīkst būt:

- vizuāls haoss,
- pārmērīgas izvēles,
- psiholoģiska pārslodze.

---

# 8. Pirmās Izvēles

Cilvēkam intuitīvi jāsaprot:

Pirmās izvēles jāierobežo līdz:

```text
pašam svarīgākajam
```


# 10. Creator Path

Cilvēkam ļoti ātri jāsaprot:

```text

```

---

```text

```

```text

ļoti dziļš kodols
```
