# ◉SYNTERRA CORE SYSTEM COMPONENTS

## Mērķis

Šis dokuments definē:

```text
SYNTERRA galvenos sistēmu komponentus
un to savstarpējo lomu infrastruktūrā
```

Tas ir pirmais pilnais sistēmu mezglu kartējums pārejā no:

```text
civilizācijas arhitektūras
uz reālu sistēmu inženieriju
```

---

# 1. Galvenais Princips

SYNTERRA netiek būvēta kā:

- viena monolīta aplikācija,
- nejaušu funkciju kopums,
- vai haotisks servisu slānis.

SYNTERRA tiek projektēta kā:

```text
modulāra dzīvas digitālās civilizācijas infrastruktūra
```
