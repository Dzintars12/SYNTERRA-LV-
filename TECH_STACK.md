# MVP STACK

Pirmajam MVP ieteicamais stack:

- Frontend: Next.js + React
- Backend: Node.js + NestJS
- Database: PostgreSQL
- Auth: Supabase / Clerk
- Realtime: WebSockets
- AI: OpenAI API
- Hosting: Vercel + Supabase / Cloud
- Storage: Supabase Storage / S3
- Analytics: PostHog / custom analytics
- Version control: GitHub

Mērķis:
ātri izveidot dzīvu, testējamu un paplašināmu pamatu.
